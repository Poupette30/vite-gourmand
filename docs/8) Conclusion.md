# Conclusion

Le projet **Vite & Gourmand** m'a permis de mettre en pratique les compétences acquises au cours de ma formation de Développeur Web et Web Mobile.

Au cours de sa réalisation, j'ai développé une application web complète en utilisant une architecture MVC, le langage PHP, une base de données relationnelle MySQL ainsi qu'une base de données NoSQL MongoDB pour les statistiques.

Ce projet m'a permis d'approfondir plusieurs notions essentielles telles que :

* la conception d'une architecture logicielle ;
* la gestion des bases de données ;
* le développement d'interfaces utilisateur ;
* l'authentification et la gestion des rôles ;
* la sécurisation des formulaires et des données ;
* l'organisation d'un projet web complet.

La réalisation de ce projet m'a également amenée à résoudre différents problèmes techniques, à rechercher des solutions adaptées et à améliorer progressivement la qualité de l'application. Cette démarche m'a permis de renforcer ma méthode de travail, ma capacité d'analyse et ma compréhension des bonnes pratiques de développement.

Si ce projet devait évoluer, plusieurs améliorations pourraient être envisagées, notamment :

* la mise en place d'un système de paiement en ligne ;
* l'ajout d'un espace de gestion des menus pour les administrateurs ;
* la création de statistiques plus détaillées ;
* l'envoi de notifications automatiques aux utilisateurs concernant leurs commandes ;
* le développement d'une interface encore plus riche et personnalisable.

La réalisation de **Vite & Gourmand** constitue une expérience concrète qui m'a permis de consolider mes connaissances techniques et de mieux comprendre les différentes étapes de conception et de développement d'une application web. Ce projet représente une étape importante dans mon parcours vers le métier de développeuse web.
