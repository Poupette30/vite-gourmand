# Fonctionnalités de l'application

## Introduction

L'application **Vite & Gourmand** propose un ensemble de fonctionnalités destinées aux clients, aux employés et aux administrateurs. Les droits d'accès sont gérés en fonction du rôle de chaque utilisateur afin de garantir la sécurité et le bon fonctionnement de l'application.

---

# Fonctionnalités accessibles à tous

Les visiteurs peuvent :

* consulter la page d'accueil ;
* parcourir la liste des menus disponibles ;
* consulter le détail d'un menu ;
* effectuer des recherches et filtrer les menus selon différents critères (prix, thème, régime alimentaire, nombre de personnes) ;
* consulter les avis validés des clients ;
* utiliser le formulaire de contact.

---

# Gestion des comptes

L'application permet :

* la création d'un compte utilisateur ;
* la connexion sécurisée ;
* la déconnexion ;
* la récupération d'un mot de passe oublié grâce à un lien de réinitialisation ;
* la réinitialisation du mot de passe.

---

# Fonctionnalités utilisateur

Après authentification, un utilisateur peut :

* consulter les menus disponibles ;
* passer une commande ;
* consulter l'historique de ses commandes ;
* suivre l'état de ses commandes ;
* laisser un avis lorsqu'une commande est terminée.

---

# Gestion des commandes

Le système permet :

* la création d'une commande ;
* le calcul automatique du prix total ;
* l'application des éventuelles remises prévues ;
* le calcul des frais de livraison ;
* le suivi des différents états de la commande.

---

# Gestion des avis

Les utilisateurs peuvent publier un avis après une commande terminée.

Chaque avis est :

* enregistré dans la base de données ;
* placé en attente de validation ;
* validé ou refusé par un administrateur avant sa publication.

---

# Fonctionnalités employé

Les employés disposent d'un espace dédié leur permettant :

* de consulter les commandes ;
* de modifier leur état d'avancement (par exemple : commande reçue, en préparation, expédiée ou terminée).

---

# Fonctionnalités administrateur

L'administrateur possède les droits les plus élevés.

Il peut notamment :

* accéder au tableau de bord d'administration ;
* créer un compte employé ;
* activer ou désactiver un employé ;
* modérer les avis des utilisateurs ;
* consulter les statistiques commerciales issues de MongoDB.

---

# Statistiques

Le tableau de bord administrateur affiche notamment :

* le nombre de commandes par menu ;
* le chiffre d'affaires réalisé ;
* la période concernée.

Ces informations sont stockées dans MongoDB afin d'illustrer l'utilisation d'une base de données NoSQL.

---

# Sécurité

L'application met en œuvre plusieurs mécanismes de sécurité :

* authentification des utilisateurs ;
* gestion des rôles (utilisateur, employé, administrateur) ;
* protection des mots de passe grâce au hachage ;
* protection contre les attaques CSRF ;
* validation des données saisies ;
* contrôle des droits d'accès aux différentes pages.

---

# Responsive Design

L'interface a été développée afin de s'adapter aux différentes tailles d'écran :

* ordinateur ;
* tablette ;
* smartphone.

L'objectif est d'offrir une expérience utilisateur cohérente quel que soit le support utilisé.

---

# Conclusion

L'application **Vite & Gourmand** propose une gestion complète d'un service de restauration en ligne. Elle couvre l'ensemble du cycle de vie d'une commande, depuis la consultation des menus jusqu'au suivi de la livraison et au dépôt d'un avis, tout en intégrant une gestion des utilisateurs, un espace d'administration et des statistiques commerciales.
