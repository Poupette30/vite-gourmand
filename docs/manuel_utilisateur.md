Version : 1.0

Application : Vite & Gourmand

Ce document explique le fonctionnement de l'application Vite & Gourmand selon les différents profils d'utilisation.

## Parcours visiteur
1. Consulter la page d’accueil.
2. Accéder à la liste des menus.
3. Filtrer par prix, thème, régime ou nombre de personnes.
4. Voir le détail d’un menu.
5. Créer un compte pour commander.

Le visiteur peut consulter librement les menus avant de créer un compte.

## Parcours utilisateur
1. Se connecter.
2. Choisir un menu.
3. Renseigner les informations de commande.
4. Valider la commande.
5. Consulter son historique.
6. Suivre les statuts.
7. Déposer un avis lorsque la commande est terminée.

## Parcours employé
1. Se connecter avec un compte employé.
2. Filtrer les commandes par statut ou client.
3. Mettre à jour les statuts.
4. Annuler une commande uniquement après contact client et saisie du motif.
5. Modérer les avis.

L'employé assure le suivi opérationnel des commandes.

## Parcours administrateur
1. Créer un compte employé.
2. Désactiver un employé.
3. Accéder à l'ensemble des fonctionnalités de l'application.
4. Consulter les statistiques par menu depuis MongoDB.

L'administrateur dispose de l'ensemble des droits de gestion de l'application.

## Bonnes pratiques
- Se déconnecter après utilisation.
- Utiliser les comptes correspondant à son rôle.
- Vérifier les informations saisies avant de valider une commande.
- Utiliser un navigateur récent (Chrome, Edge ou Firefox).
